trastorno específico del aprendizaje que afecta la capacidad para comprender, aprender y utilizar conceptos matemáticos. No se debe a falta de inteligencia ni a una enseñanza inadecuada, sino a dificultades en el procesamiento numérico y el razonamiento matemático.
## Caracteristicas
 - Dificultad para reconocer y comprender números.
 - Problemas para realizar operaciones matemáticas básicas.
 - Confusión con los signos matemáticos (+, -, ×, ÷).
 - Dificultad para memorizar tablas de multiplicar.
 - Problemas para comprender secuencias numéricas.
 - Dificultad para manejar conceptos de tiempo, dinero y medidas. 
## Tipos
 - Discalculia Verbal: dificultad para nombrar cantidades y conceptos matemáticos.
 - Discalculia Practognóstica: dificultad para manipular y comparar cantidades.
 - Discalculia Léxica: problemas para leer símbolos y números.
 - Discalculia Gráfica: dificultades para escribir números y símbolos matemáticos.
 - Discalculia Operacional: dificultades para realizar cálculos y operaciones matemáticas. 
## Estrategias
 - Utilizar material concreto y manipulativo.
 - Enseñar conceptos matemáticos de forma visual.
 - Relacionar las matemáticas con situaciones cotidianas.
 - Dividir las tareas en pasos pequeños y secuenciales.
 - Reforzar constantemente los logros obtenidos.
 - Utilizar juegos educativos que impliquen números y cantidades.
